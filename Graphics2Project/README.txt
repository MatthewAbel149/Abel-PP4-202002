Controls:

Camera:
	W	: Forward
	S	: Backward
	A	: Left
	D	: Right

	SPACE	: Up
	C	: Down

	O	: Decrease camera speed
	P	: Increase camera speed

	MOUSE	: Rotate camera

	R	: Reset camera
	ESC	: Toggle cursor visibility

Scene:
	N	: Next scene
Misc:
	B	: (Scene 0) Enlarge king
	K	: (Scene 0) Rotate king
